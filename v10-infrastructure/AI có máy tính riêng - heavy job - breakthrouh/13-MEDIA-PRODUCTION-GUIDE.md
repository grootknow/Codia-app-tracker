## üé¨ CONTENT PRODUCTION WORKFLOW

### **Stage 1: Pre-Production (2-5 days)**

```yaml
Ideation:
  Tools: Notion board, AI research agents
  Process:
    1. Brainstorm topic ideas
    2. Research trends (TubeBuddy, Google Trends)
    3. Competitor analysis (what's working?)
    4. Select best idea (views potential + alignment)
  Output: Topic decision

Research:
  Tools: AI agents (Claude, Perplexity), web search
  Process:
    1. Deep dive on topic
    2. Gather data, stats, examples
    3. Find expert quotes
    4. Compile resources
  Output: Research doc (Notion)

Scriptwriting:
  Tools: Google Docs, AI assistance (Claude)
  Process:
    1. Outline (hook, intro, body, CTA)
    2. Write first draft
    3. AI feedback for clarity
    4. Edit for pacing, storytelling
    5. Finalize script
  Output: Final script (800-1200 words for 8-10 min video)

Storyboard:
  Tools: Canva, Figma
  Process:
    1. Shot list (what to film)
    2. Visual references
    3. B-roll requirements
    4. Graphics/animations needed
  Output: Storyboard document

Equipment Prep:
  Checklist:
    ‚òê Charge all batteries (camera, lights, mics)
    ‚òê Format SD cards
    ‚òê Pack equipment (use checklist)
    ‚òê Test audio levels
    ‚òê Prepare set/location
```

### **Stage 2: Production (1-2 days)**

```yaml
Setup:
  Camera:
    - Mount on tripod
    - Set manual exposure (avoid auto)
    - White balance (custom for location)
    - Frame rate: 24fps (cinematic) or 60fps (smooth)
    - Resolution: 4K (future-proof)
    
  Lighting:
    - Key light: 45¬∞ angle, soft (through softbox)
    - Fill light: Opposite side, 1/2 intensity
    - Back light: Separate subject from background
    - Test with camera, adjust
    
  Audio:
    - Attach lav mic to talent
    - Boom mic as backup
    - Test levels: -12dB to -6dB range
    - Monitor with headphones

Filming:
  A-Roll (main footage):
    - Record in takes (easier to edit)
    - Clap or clapper for sync
    - Multiple takes per segment
    - Check focus, exposure after each take
    
  B-Roll (supporting footage):
    - Follow shot list from storyboard
    - Get variety (wide, medium, close-up)
    - 10-15 seconds per clip minimum
    - Smooth gimbal/slider shots
    
  Quality Checks:
    ‚òê Focus sharp
    ‚òê Exposure correct (not clipped)
    ‚òê Audio clean (no background noise)
    ‚òê Framing on point
    ‚òê Multiple takes of key moments

Backup:
  Immediate:
    1. Copy files from SD to NAS (hot tier)
    2. Verify files copied correctly
    3. Keep SD cards until project done
  
  Same Day:
    4. Auto-backup NAS ‚Üí R2 (every 4 hours)
    5. Verify in R2 bucket
```

### **Stage 3: Post-Production (3-7 days)**

```yaml
Editing (Premiere Pro):
  Day 1-2: Assembly
    1. Import footage to Premiere
    2. Create project structure (bins)
    3. Rough cut: Assemble clips per script
    4. Audio sync (if separate mic)
    5. Remove bad takes, long pauses
  
  Day 3-4: Fine Cut
    1. Trim for pacing (cut tight!)
    2. Add B-roll inserts
    3. Insert graphics placeholders
    4. Add music (from Artlist)
    5. Add SFX where needed
  
  Day 5: Graphics
    1. Create title cards (After Effects)
    2. Lower thirds (name, title)
    3. Transitions (subtle, not cheesy)
    4. End screen (subscribe, next video)

Color Grading (DaVinci Resolve):
  Process:
    1. Export XML from Premiere
    2. Import to Resolve
    3. Apply LUT (Sony S-Log3 ‚Üí Rec709)
    4. Color correct (balance, exposure)
    5. Color grade (style, mood)
    6. Export back to Premiere

Audio Mixing (Audition):
  Process:
    1. Remove background noise (adaptive)
    2. EQ for clarity (cut low freq, boost presence)
    3. Compression (even out levels)
    4. Normalize to -3dB peak
    5. Add music (duck under dialogue)
    6. Master limiter (prevent clipping)

Final Export:
  Settings:
    - Format: H.264 (MP4)
    - Resolution: 3840x2160 (4K)
    - Frame rate: Same as source
    - Bitrate: 40-50 Mbps (VBR 2-pass)
    - Audio: AAC 320kbps stereo
  
  Versions:
    - YouTube: 4K full version
    - TikTok: 1080x1920 vertical, <60s cuts
    - Instagram: 1080x1080 or 1080x1920, <90s
```

### **Stage 5: Analytics & Optimization (Ongoing)**

```yaml
Track (First 48 hours):
  Metrics:
    - Views (velocity matters)
    - CTR (click-through rate on thumbnail/title)
    - AVD (average view duration) - aim >50%
    - Engagement (likes, comments, shares)
    - Traffic sources (where views coming from)
  
  Tools:
    - YouTube Studio (real-time)
    - TubeBuddy (competitor benchmarking)
    - Custom dashboard in tracker app

Analyze (Week 1):
  Questions:
    - What's working? (high retention parts)
    - Where dropoff? (edit tighter next time)
    - Which thumbnail won A/B test?
    - Best traffic source?
    - Subscriber conversion rate?
  
  Actions:
    - Comment responses (engagement algo boost)
    - Pin top comment (context or CTA)
    - Update description if needed
    - Promote on other platforms

Calculate ROI:
  Formula:
    Production Cost = Equipment depreciation + Software + Time + Outsource
    Revenue = Ad revenue + Sponsorships + Affiliate
    ROI% = ((Revenue - Cost) / Cost) √ó 100
  
  Track in Database:
    - content_production_costs table
    - content_revenue table
    - Calculate per video, per month, per channel

Iterate (Month 1+):
  What to optimize:
    - Topics (double down on high performers)
    - Thumbnails (test variants)
    - Titles (SEO + curiosity)
    - Intro hook (first 30 seconds critical)
    - CTA placement (when to ask subscribe)
    - Publishing schedule (consistency > perfection)
  
  Goals:
    - Month 1: Publish 8 videos (2x/week)
    - Month 3: 100 subscribers
    - Month 6: 1,000 subscribers (monetization)
    - Month 12: 10,000 subscribers
    - Revenue target: $500/mo @ 1K subs, $5K/mo @ 100K subs
```
